Welcome to Lando

Lando is an automated code lander for Mozilla. It is integrated with our Phabricator instance and can be used to land revisions to various repositories.

Each revision in Phabricator has a "View Stack in Lando" link in its action menu on the right. This will take you to the Lando page associated with that revision. If a revision cannot be landed, either due to its current state, or due to it not being configured in or supported by Lando, the link will be greyed out, but you can still follow it to view its details in Lando.

For more information on Lando, consult the user guide.
